# PO-kolokwia
Rozwiązania do kolokwiów z programowania obiektowego z poprzednich lat MIMUW.

# Instrukcja korzystania
Stwórz folder (jeśli nie istnieje) z nazwą roku z którego jest kolokwium, następnie umieść rozwiązanie w podfolderze ze swoim imieniem (lub inicjałami, itd). Najlepiej, aby rozwiązanie było przepisane na komputer, ale może być zdjęcie.
